Su programa/script bash debe recibir el nombre del código a compilar cómo
parámetro. Una vez recibido debe compilarlo y ejecutar las pruebas correspondientes
con los archivos de prueba que se encuentran en la carpeta inputs.

El output de estos debe almacenarse en una carpeta llamada outputs.
Esta también debe ser creada por su script.

Una vez finalizada las pruebas muestre el contenido de los outputs generados
usando el comando cat.

CUIDADO!!
Revise que esté recorriendo correctamente los casos de prueba y los
archivos de output tengan la enumeración de su archivo de input
correspondiente.
